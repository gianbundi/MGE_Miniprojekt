# Score Counter

Für das Mini Projekt wird ein Score Counter erstellt, der folgende Features beinhalten soll:

- Neues Scoreboard anlegen / bestehendes nutzen
- Spieler hinzufügen
- Anzahl Punkte/Runde eingeben
- Resultate automatisch ausrechnen
- Theme Switcher
- evtl. noch weiteres...
